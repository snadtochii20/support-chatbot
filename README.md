#  Chatbot OpenAI

## Опис
Веб-чатбот для текстової взаємодії користувача з AI з можливістю збереження історії повідомлень у Firebase Firestore.

---

## Використані технології
- Node.js  
- JavaScript  
- HTML / CSS  
- Bootstrap  
- Firebase Firestore
